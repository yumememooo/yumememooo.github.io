java spring 非同步事件

Posted on 2021-01-30 In Tech. , back-end , java Views:

spring 加上 Fire and forget，非同步處理，發出處理後就不用等待回復繼續做其他事情

1.找到@Configuration 的地方加上@EnableAsync

2.找到想要執行非同步的方法上方加上 @Async

如果想要拿到回傳的地方可以在 Future拿到結果。
